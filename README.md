### ArtistFinder

A simple React app made for as a final hand-in for the frontend development class at Teknikhögskolan. It's based on a previous assignment which was made with traditional HTML/CSS/JavaScript, which I've now ported over to React. 

The app connects to an API that our teacher created, and fetches info on a number of (fictional) bands which are displayed in a list. There's also a Details page with additional info like albums and members, and an Add New page where you can submit your own music group by picking a name, genre and est. year. Using the "Seeded" checkbox you can toggle between seeing bands randomly generated by the teacher, and bands that students themselves have uploaded to the API through their own apps.
